====================================================================
                   1. PREREQUISITES
====================================================================

- Java 11 (OpenJDK or Oracle JDK)
- VS Code with these extensions:
    * Extension Pack for Java (by Microsoft)
    * (Optional) Maven for Java
- Internet connection (dependencies total ~200 MB)

====================================================================
                   2. COMMANDS TO RUN
====================================================================

Open a terminal in VS Code (Ctrl+`) and run one of these:

A) If Maven is installed globally:
    mvn clean install
    mvn dependency:resolve      (to only download dependencies)
    mvn spring-boot:run         (to start the application)

B) Using the Maven Wrapper (included in project):
    Windows:   mvnw.cmd clean install
    macOS/Linux: ./mvnw clean install
    (Replace "clean install" with "dependency:resolve" or "spring-boot:run")

C) Via VS Code Command Palette:
    Ctrl+Shift+P → "Maven: Execute Commands" → choose "mvn install"

If dependencies don't load:
    Ctrl+Shift+P → "Java: Clean Java Language Server Workspace" → Reload VS Code

====================================================================
                   3. DEPENDENCY LIST (all libraries)
====================================================================

Spring Boot:
  - spring-boot-starter-web        (REST APIs)
  - spring-boot-starter-security   (security, JWT)
  - spring-boot-starter-validation (input validation)

AWS SDK v2:
  - s3                             (file storage)
  - dynamodb                       (NoSQL database)
  - cognitoidentityprovider        (user authentication)
  - apigateway                     (API Gateway client)
  - sns                            (notifications)

JWT:
  - jjwt-api, jjwt-impl, jjwt-jackson (JSON Web Tokens)

NLP & AI (Java):
  - stanford-corenlp 4.5.1         (NER, POS tagging)
  - opennlp-tools 2.3.2            (alternative NER)
  - deeplearning4j-core 1.0.0-M2.1 (deep learning / BERT)
  - nd4j-native-platform 1.0.0-M2.1 (numpy-like tensors)
  - deeplearning4j-nlp 1.0.0-M2.1  (NLP tools)
  - djl-api & model-zoo 0.27.0     (Hugging Face model loading)

Data Processing:
  - tablesaw-core 0.43.0           (dataframes, like pandas)
  - commons-math3 3.6.1            (mathematical operations)

File Parsing:
  - pdfbox 3.0.1                   (read PDF resumes)

Utilities:
  - lombok 1.18.30                 (reduce boilerplate)

Testing:
  - spring-boot-starter-test       (unit & integration tests)
